# Security

Velocitron 0.1.0 is an alpha evaluation release. It has no formal security
support program, service-level agreement, response-time commitment, or
currently supported private vulnerability-reporting channel. Do not use it in
production, security-sensitive, or otherwise high-consequence contexts.

If you received this source through the soft launch and already have a direct
channel with the author, you may use that existing channel to share a suspected
vulnerability. This is not a promise of response, remediation, disclosure
timing, or ongoing support.

If you do not already have such a channel, there is currently no supported
private reporting route. Please avoid publicly disclosing exploit details or
instructions that would facilitate abuse. A public report may state that a
potential security issue exists without including actionable exploitation
details.
